# RAPTOR AI — Система детекции военных объектов на изображениях и видео

## Описание проекта

**RAPTOR AI** — это современная система для автоматического распознавания и классификации военных объектов на фотографиях и видео. Проект состоит из двух частей:
- **Backend** (FastAPI, Python): обработка изображений/видео, инференс моделей YOLO, API для фронта.
- **Frontend** (React + Vite): удобный интерфейс для загрузки файлов, просмотра результатов, управления моделями.

### Основной функционал

- Загрузка изображений и видео (drag&drop, мультизагрузка до 5 файлов).
- Автоматический запуск инференса для каждого файла.
- Просмотр результатов с наложением детекций.
- Скачивание результатов (JPG, JSON).
- Уведомления об ошибках, лимитах, статусах.
- Аутентификация через Auth0 (опционально).
- Локализация интерфейса (украинский, английский).
- Адаптивный дизайн.

### Стадия разработки

- [x] MVP: базовая загрузка, инференс, отображение результатов.
- [x] Ограничение на количество файлов, обработка ошибок.

## Быстрый старт (для новичков)

### 1. Клонирование репозитория

```sh
git clone https://github.com/somedumbslime/cv_site.git
cd cv_site
```

### 2. Установка зависимостей

#### Backend (Python, FastAPI)

- Требуется установленный Python 3.9+

```sh
cd backend
python -m venv ИМЯ_ВИРТ_ОКРУЖЕНИЯ
source ИМЯ_ВИРТ_ОКРУЖЕНИЯ/bin/activate  # для Windows: venv\Scripts\activate
pip install --upgrade pip
pip install -r requirements.txt
```

#### Frontend (React + Vite)

- Требуется установленный [Node.js](https://nodejs.org/) (рекомендуется LTS-версия)

```sh
cd ../frontend
npm install
```

### 3. Переменные окружения

#### Backend

Создайте файл `.env` в папке `backend/` и укажите:

```
AUTH0_DOMAIN=your-auth0-domain
API_AUDIENCE=your-api-audience
```

#### Frontend

Создайте файл `.env` в папке `frontend/` и укажите:

```
VITE_AUTH0_DOMAIN=your-auth0-domain
VITE_AUTH0_CLIENT_ID=your-auth0-client-id
VITE_AUTH0_AUDIENCE=your-auth0-audience (опционально)
```

### 4. Запуск проекта

#### Backend

```sh
cd backend
uvicorn main:app --reload
```

или с поддержкой внутри одной Wi-Fi сети

```sh
cd backend
uvicorn main:app --host 0.0.0.0 --port 8000 --ssl-keyfile=../frontend/certs/localhost-key.pem --ssl-certfile=../frontend/certs/localhost.pem
```

#### Frontend

```sh
cd frontend
npm run dev 
```

или с поддержкой внутри одной Wi-Fi сети

```sh
cd frontend
npm run dev -- --host
```

Откройте [http://localhost:5173](http://localhost:5173) в браузере.

ВАЖНО! в файлах указан мой персональный ip-адресс, он может отличаться от вашего, его нужно будет узнать и поменять в файлах
